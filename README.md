**MD International**

MD International is a group of construction companies providing comprehensive solutions in infrastructure, contracting, and EPC projects.
This repository contains the complete source code of the company’s website, built using PHP, HTML, CSS, and JavaScript.
**MD-International/**
│
├── Home/                # Homepage
├── About/               # About Us page
├── Contact/             # Contact page
├── Products/            # Products listing page
├── Product/             # Single product details
├── Layout/              # Shared layout components
├── images/              # Images and visual assets
├── MD International.zip # Zipped version of the project
└── README.md            # Documentation file
**Website Features**
Modern and responsive design
